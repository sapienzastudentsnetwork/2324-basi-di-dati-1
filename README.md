# Basi di Dati - I modulo

Lo scopo di questo repository GitHub è la condivisione e discussione delle soluzioni agli esercizi (inclusi quelli tratti da tracce di prove di esame passate) di Basi di Dati I forniti dalla prof.ssa Maria de Marsico e dal prof. Giuseppe Perelli. In questo repository potrai dunque trovare (e possibilmente confermare) soluzioni proposte da altri studenti o anche condividere e ricevere un feedback in merito alle tue!
